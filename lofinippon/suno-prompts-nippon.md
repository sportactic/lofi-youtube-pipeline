# LoFi Nippon — Suno Prompt Templates

## Model
**Always use Suno v5.5 (chirpfenix)** — same as @The-Lo-Fi for quality, but with Japanese instrument focus.

## Generation Rules
- Each prompt → 2 variations = 2 tracks
- 5 prompts × 2 = 10 tracks per session
- Each track ~120-190s
- Total ~22-25 min × 2 loops = 45-50 min
- Or 5 prompts × 2 × 3 loops = 68-75 min for 1+ hour

## Style Descriptors (variations)

### Mood modifiers
- `peaceful` — calm, slow
- `melancholic` — sad, wistful
- `nostalgic` — memory-evoking
- `dreamy` — ethereal, soft
- `warm` — cozy, intimate
- `hopeful` — uplifting
- `mysterious` — foggy, distant
- `meditative` — mindful, breathing

### Instrument focus
- `koto-heavy` — emphasize string plucks
- `shakuhachi-solo` — flute lead
- `shamisen-driven` — plucked string rhythm
- `piano-and-koto` — both lead
- `taiko-pulse` — drum-driven

## Quality Notes
- Always include `instrumental, no vocals` (lo-fi audience prefers no vocals)
- Always specify BPM (60-85 range for chill)
- Include specific ambience (rain, wind chimes, footsteps, etc.)
- Keep prompts under 200 words

## Workflow
1. User logs into Suno.com (their account)
2. Use one of the prompts above, generate 2 variations each = 10 tracks
3. Wait for all 10 to finish
4. Agent captures /me page → downloads audio
5. Proceed to visuals + render
